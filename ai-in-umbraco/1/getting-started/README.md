# Getting Started

The Getting Started guide walks you through installing Umbraco.AI, configuring your first AI connection, and making your first AI-powered request.

## Prerequisites

Before you begin, ensure you have:

- Umbraco CMS 17.0 or later installed
- .NET 10.0 SDK or later
- An API key from your AI provider (for example, OpenAI)
