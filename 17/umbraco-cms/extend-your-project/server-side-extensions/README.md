# Server-side Extensions

Server-side extensions allow you to go beyond the Backoffice UI, giving you direct control over the underlying .NET framework and Umbraco’s internal services.

Whether you are building real-time features with SignalR or integrating third-party services via custom API controllers, these guides provide the architecture needed to build robust, enterprise-scale applications.

### How to use this section

The following articles are a curated collection of independent guides focused on different aspects of the Umbraco back-end. While these topics aren't necessarily related to one another in a linear path, they each provide the technical foundation for a specific extension point.
