# Actions

The Commerce add-on contributes the following actions.

## Order Read Actions

| Display Name | Alias |
| ------------ | ----- |
| Get Order | `umbracoCommerce.getOrder` |
| Search Orders | `umbracoCommerce.searchOrders` |
| Search Discounts | `umbracoCommerce.searchDiscounts` |
| Search Gift Cards | `umbracoCommerce.searchGiftCards` |

## Order Modify Actions

| Display Name | Alias |
| ------------ | ----- |
| Add Product to Order | `umbracoCommerce.addProductToOrder` |
| Remove Order Line | `umbracoCommerce.removeOrderLine` |
| Assign Order to Customer | `umbracoCommerce.assignOrderToCustomer` |
| Change Order Status | `umbracoCommerce.changeOrderStatus` |
| Finalize Order | `umbracoCommerce.finalizeOrder` |
| Set Order Property | `umbracoCommerce.setOrderProperty` |
| Tag Order | `umbracoCommerce.tagOrder` |
| Remove Tags from Order | `umbracoCommerce.removeTagsFromOrder` |
| Export Order | `umbracoCommerce.exportOrder` |
| Send Order Email | `umbracoCommerce.sendOrderEmail` |

## Payment Actions

| Display Name | Alias |
| ------------ | ----- |
| Capture Payment | `umbracoCommerce.capturePayment` |
| Refund Payment | `umbracoCommerce.refundPayment` |

## Stock and Gift Card Actions

| Display Name | Alias |
| ------------ | ----- |
| Update Stock | `umbracoCommerce.updateStock` |
| Issue Gift Card | `umbracoCommerce.issueGiftCard` |

{% hint style="info" %}
Open the step in the canvas to see each action's settings and output fields.
{% endhint %}
