# AI

The AI add-on bridges Umbraco Automate and [Umbraco.AI](https://docs.umbraco.com/ai-in-umbraco). It adds triggers that fire on AI agent lifecycle events and actions that let an automation invoke an AI agent or transcribe audio.

## What It Adds

| Type | Items |
| ---- | ----- |
| Triggers | AI Agent Run Completed, AI Agent Run Failed, AI Agent Request |
| Actions | Run AI Agent, Transcribe Audio |
