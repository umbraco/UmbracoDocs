# Engage

The Engage add-on lets your automations react to Umbraco Engage events and act on goals, persona scores, and customer journey steps.

## What It Adds

| Type | Items |
| ---- | ----- |
| Triggers | A/B test lifecycle, sessions, pageviews, segments, personas, customer journeys, goals, campaigns |
| Actions | Trigger Goal, Score Persona, Score Customer Journey Step |
