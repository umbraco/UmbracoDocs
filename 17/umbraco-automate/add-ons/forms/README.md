# Forms

The Forms add-on lets your automations react to form submissions and approvals, and lets you submit or export form entries from inside an automation.

## What It Adds

| Type | Items |
| ---- | ----- |
| Triggers | Form Submitted, Form Entry Approved |
| Actions | Submit Form, Export Form Entries |
