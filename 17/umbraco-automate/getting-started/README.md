# Overview

The Getting Started guide walks you through installing Umbraco Automate, applying the required configuration, and building your first automation.

## Prerequisites

Before you begin, make sure you have:

* Umbraco CMS 17.1 or later installed.
* .NET 10.0 SDK or later.
* Permission to add NuGet packages and edit `appsettings.json` .
