# Frontend Tools

{% hint style="warning" %}
Frontend tools are part of the [`Umbraco.AI.Agent.Copilot`](../agent-copilot/README.md) package. See the [Agent Copilot frontend tools documentation](../agent-copilot/frontend-tools.md) for full details.
{% endhint %}

Frontend tools allow agents to perform actions in the browser. When an agent calls a tool, the frontend receives the call and executes the action. This is handled by the Umbraco.AI.Agent.Copilot package.

See [Frontend Tools](../agent-copilot/frontend-tools.md) for the full guide including tool definitions, handlers, and common patterns.
