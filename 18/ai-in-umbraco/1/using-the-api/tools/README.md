# Tools

Tools let AI models call functions during a conversation. The model decides when to use a tool based on the user's request, and tool results are fed back into the conversation.

Umbraco AI supports tools through Microsoft.Extensions.AI (M.E.AI), giving you a standard way to expose functionality to AI models.
