# Users and Members

**Users** have access to the Umbraco backoffice. **Members** are front-end website users who register and log in on your public site. They are separate concepts managed in different sections of the backoffice.

* [Users](users/) - create and invite backoffice users, manage User Groups, and configure section and content permissions.
* [Members](members.md) - manage front-end website members, Member Types, and Member Groups for public authentication.
